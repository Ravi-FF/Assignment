Que-1. What are the benefits of using CSS?

  2. Multiple device compatibility : With the help of CSS, we can create responsive websites for mobile, tablet, laptop and different devices.

  3. SEO friendly wesite : With the help of CSS, we can create a SEO friendly website so that our website has a good effect on the web browser.

  4. Create beautiful webite : With the help of CSS, we can create a beautiful website which looks good to the user.

  5. Faster-page loding : Inside CSS , we can change the HTML document by selecting particular elements so that our website loads faster due to less code.

Que-2. What are the disadvantages of CSS?

  1. Security issue : We get to see limited security inside CSS. Any person who has even a little knowledge of the website can see the code of any website and make changes in it.

  2. There are different levels inside CSS (CSS2 ,CSS3). which are a little difficult to understand.

  3. Browser compatibility : Some browsers do not support style sheets, due to which developers face problems in creating websites.

Que-15. What is the meaning of the CSS selector?

  • Selector : With the help of selector we can style the HTML element. There are selectors somewhere inside the CSS. for example :

    • Basic selectors

      1. Universal selector

      2. Id selector

      3. Class selector

      4. Attribute selector

      5. Element selector

    • Combinator selectors

      1. Descendant selector

      2. Child selector (>)

      3. Adjacent sibling selector (+)

      4. General sibling selector (~)

    • Group selectors

    • Pseudo selectors

      1. Pseudo Class selector (:)

      2. Pseudo element selector (::)
