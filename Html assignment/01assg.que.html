Que. 11 What is the difference between a block-level element and an inline element?

block level element = Block level elements are those elements that take up the full width of the web browser and display the next content in the next line.we can set properties like width, height, margin, and padding in block level element. For example = < div >, < ul > , < dl > , < p > ,< h1 to h6 > etc.
inline level element = inline elements are those elements that take up as much width as the content requires and display the next content in the same line without showing it in the next line. can not set any properties like width, height, margin, and padding in inline element. For example = < hr >, < img > , < br > , < a > etc.
